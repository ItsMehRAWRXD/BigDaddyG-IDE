## 🏗️ Architecture: Complete Compiler Framework

### Full Compiler Pipeline in PowerShell

```powershell

Source Code
    ↓
┌─────────────────────────────────────┐
│ LEXICAL ANALYSIS (Tokenizer)       │
│ - Scan characters                   │
│ - Generate tokens                   │
│ - Identify keywords, operators      │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ PARSING (Recursive Descent)         │
│ - Build Abstract Syntax Tree (AST)  │
│ - Validate syntax                   │
│ - Check grammar rules               │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ SEMANTIC ANALYSIS                   │
│ - Type checking                     │
│ - Symbol table                      │
│ - Scope resolution                  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ CODE GENERATION                     │
│ - Byte-level assembly               │
│ - Optimization passes               │
│ - Target-specific code              │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ BINARY GENERATION                   │
│ - PE file format (Windows)          │
│ - ELF format (Linux)                │
│ - Mach-O format (macOS)             │
└─────────────────────────────────────┘
    ↓
Executable Binary

```plaintext
