## 🪄 **NEW: AI COMMAND GENERATOR** (Just Added!)

### **What It Does:**

- Generates terminal commands from natural language
- Respects safety levels
- Shows each command with Run button
- Can execute all at once
- Copies to clipboard

### **How to Use:**

```plaintext
1. Press: Ctrl+Shift+G
2. Type: "Install Express.js and create basic server"
3. AI generates:

   • npm init -y
   • npm install express
   • New-Item server.js
   • (Creates server.js code)

  1. Click "Execute All" or run individually
  2. Done!
```plaintext
### **Example Requests:**

**Package Installation:**

```plaintext
"Install React, React Router, and Axios"

Generated:
→ npm install react react-dom
→ npm install react-router-dom
→ npm install axios

```plaintext
**Git Operations:**

```plaintext
"Commit all changes and push to main branch"

Generated:
→ git add .
→ git commit -m "Update: $(date)"
→ git push origin main

```plaintext
**File Search:**

```plaintext
"Find all Python files with TODO comments"

Generated:
→ Get-ChildItem -Recurse -Filter *.py | Select-String "TODO"

```plaintext
**Build & Deploy:**

```plaintext
"Build Docker image and deploy to production"

Generated:
→ docker build -t myapp:latest .
→ docker tag myapp:latest registry.com/myapp:latest
→ docker push registry.com/myapp:latest
→ kubectl apply -f deployment.yaml

```plaintext
**Safety Check:**

- **SAFE mode:** Asks before each command
- **BALANCED mode:** Asks for destructive commands
- **YOLO mode:** Executes everything automatically
